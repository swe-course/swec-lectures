# Environment for Continues Integration

## Prerequisites
* Allocate Ubuntu VM (18.04)
* Install helper - [talan cli](https://github.com/project-talan/tln-cli)
* Install docker
  ```
  tln install docker:docker-compose
  ```
 * Clone repository with lectures
  ```
  cd ~
  mkdir projects
  cd projects
  mkdir swe-course
  cd swe-course
  git clone 
  ```
